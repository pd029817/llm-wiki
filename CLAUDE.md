# LLM-Wiki 지식관리 시스템

## 폴더 구조
- `sources/` — 원본 문서 (웹페이지 저장, 워드, PDF 등)
- `raw/` — 마크다운 변환된 원본
- `wiki/` — LLM이 생성한 위키 페이지들
- `app/` — 웹앱 코드 (질의 인터페이스)

## 작업 규칙
- 원본 문서 보존 (sources/ 폴더)
- 마크다운 변환 시 원본 출처 URL/파일명 메타데이터 필수
- 위키 페이지에 "최종 업데이트 날짜" 표시
- 한국어로 작성

## 하지 마
- 원본 문서 삭제/수정 금지
- 개인정보가 포함된 문서는 마스킹 처리 후 등록
